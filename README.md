## ğŸš€ Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/epulse.git
   
   cd epulse
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

